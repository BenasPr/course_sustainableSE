summary: "Browsing might use more energy than a user imagines, on the browser device itself and on the datacenters running applications. We want to implement a browser extension that would allow to enable sustainable mode. In this mode, users would be able to limit scripts running on websites, disable autoplay, automatically run videos in lower quality, disable AI in search engines etc. The extension will track an estimate of the energy usage, so users can adapt their behaviour and see the impact they make."
